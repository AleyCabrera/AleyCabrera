- 💻 Desarrollador Full Stack con pasión por la **ciberseguridad**
- 🏭 Enfocado en la **convergencia IT/OT** y arquitecturas seguras
- 🛠️ Desde **soporte técnico** hasta desarrollo: entiendo hardware, redes y sistemas
